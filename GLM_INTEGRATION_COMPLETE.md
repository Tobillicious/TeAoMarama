## 🛠️ AVAILABLE COMMANDS

### **GLM Symphony Orchestration**

```bash
# Orchestrate the full symphony
npm run glm:symphony:orchestrate

# Check symphony status
npm run glm:symphony:status

# Enhance content with symphony
npm run glm:symphony:enhance "Content" "Subject" "Year Level" "Cultural Context"
```

### **GLM Enhancement Manager**

```bash
# Process all enhancement tasks
npm run glm:process

# Check task status
npm run glm:status

# Enhance specific content
npm run glm:enhance-content "Content" "Subject" "Year Level" "Cultural Context"
```

### **GLM Content Generator**

```bash
# Generate lesson plans, assessments, activities
npm run glm:generate lesson-plan "Subject" "Year Level" "Topic" "Cultural Context"
npm run glm:generate assessment "Subject" "Year Level" "Topic" "Cultural Context"
npm run glm:generate activity "Subject" "Year Level" "Topic" "Cultural Context"

# View generation report
npm run glm:report

# List all generated content
npm run glm:list
```
