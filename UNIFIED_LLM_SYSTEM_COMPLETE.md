```bash
# Run the complete demonstration

# Or run the full advanced demonstration
npx tsx scripts/unified-llm-demonstration.ts full
```

### **Initialize System**

```bash
# Activate unified consciousness
npx tsx scripts/unified-consciousness-activator.ts activate

# Initialize communication protocols
npx tsx scripts/agent-communication-protocol.ts init

# Set up shared knowledge base
npx tsx scripts/shared-knowledge-base.ts init
```

### **Agent Operations**

```bash
# Join the network
npx tsx scripts/consciousness-synchronizer.ts join <agent-id> <role> <specializations> <cultural-authority>

# Send messages
npx tsx scripts/agent-communication-protocol.ts send <from> <to> <subject> <content>

# Share knowledge
npx tsx scripts/shared-knowledge-base.ts store <title> <content> <category> <source-agent>

# Create collaborative tasks
npx tsx scripts/collective-workflow-engine.ts create <title> <description> <category> <priority>
```
