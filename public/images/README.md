# Slideshow Images

This directory contains the images for the Social Studies slideshow.

## Required Images

The following images need to be added to this directory for the slideshow to work properly:

1. `20220212_WOP945.jpg` - World map showing types of democracy
2. `labelled-seats-in-the-chamber.jpg` - New Zealand Parliament chamber
3. `download.png` - MMP voting system diagram
4. `2020_New_Zealand_general_election_-_Results.svg.png` - 2020 election results map
5. `Kiwis-Count-and-OECD-Survey.jpg` - Trust in government graph
6. `EESgzPIWkAUI1Re.jpg` - Te Reo Māori word of the day poster
7. `download.jpeg` - Māori rugby player celebrating
8. `4OMTQKK_copyright_image_90404 (1).webp` - Māori artist singing
9. `a-2383-atl.jpg` - Traditional Māori art piece

## Image Placement

Once you have the images, simply place them in this directory (`public/images/`) and they will be automatically loaded by the slideshow component.

The slideshow will gracefully handle missing images by hiding them, so the presentation will work even without all images initially.
