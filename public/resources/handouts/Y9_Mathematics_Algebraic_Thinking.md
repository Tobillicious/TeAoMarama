## Algebraic Thinking Activities

### 📊 **Activity 1: Population Pattern Recognition**

**Time**: 30 minutes | **Materials**: Calculator, graph paper, colored pencils

**Task 1: Linear Pattern Analysis** (15 minutes)
Using the Auckland population data:

1. **Find the pattern**: Calculate the annual increase for each year
2. **Write the relationship**: If P = population and t = years since 2018, write P = ?
3. **Make predictions**: What will Auckland's population be in 2030?
4. **Test your formula**: Does it match the given data?

**Task 2: Exponential Thinking** (15 minutes)
Using the total NZ population data:

1. **Calculate growth rates**: Find the percentage increase each year
2. **Look for patterns**: Is the growth rate consistent?
3. **Create expressions**: Write an expression for exponential growth
4. **Compare models**: Which is more accurate - linear or exponential?

**Extension Questions**:

- How do population growth patterns affect infrastructure planning?
- What factors might change these patterns in the future?

## Advanced Algebraic Concepts

### 🔢 **Pattern Recognition Strategies**

**Linear Patterns** (arithmetic sequences):

- **Constant difference**: Each term increases/decreases by the same amount
- **Formula**: aₙ = a₁ + (n-1)d, where d is the common difference
- **Example**: Auckland population increases by ~22,500 annually

**Exponential Patterns** (geometric sequences):

- **Constant ratio**: Each term is multiplied by the same factor
- **Formula**: aₙ = a₁ × rⁿ⁻¹, where r is the common ratio
- **Example**: EV registrations growing by varying percentages

**Quadratic Patterns**:

- **Second differences constant**: Pattern in differences of differences
- **Formula**: aₙ = an² + bn + c
- **Example**: Some environmental recovery patterns

### 📈 **Modeling Real-World Relationships**

**Direct Variation**: y = kx

- Example: Kauri height growth over time
- As time increases, height increases proportionally

**Inverse Variation**: y = k/x

- Example: Population density as area increases
- As area increases, density decreases

**Exponential Growth/Decay**: y = abˣ

- Growth: b > 1 (EV adoption)
- Decay: 0 < b < 1 (kauri dieback with treatment)
