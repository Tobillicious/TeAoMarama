# Y8 Science: Chemistry - States of Matter in New Zealand Context

**Subject**: Science  
**Year Level**: 8  
**Duration**: 50 minutes  
**Curriculum Level**: 4  
**NZC Strand**: Physical World - Properties and Changes of Matter

---

## 🎯 **WALT & WALA**

**WALT** (We Are Learning To):

- Understand the three states of matter (solid, liquid, gas) and their properties
- Explain how temperature affects state changes in everyday materials
- Investigate state changes using New Zealand examples and contexts
- Apply scientific understanding to real-world phenomena

**WALA** (We Are Learning About):

- The particle theory of matter and how it explains state changes
- Melting, freezing, evaporation, and condensation processes
- How state changes are important in New Zealand's environment and industry
- Scientific investigation and experimental design

---

## 📚 **CONTENT STRUCTURE**

### **Particle Theory Review (10 minutes)**

- **Solids**: Fixed shape, fixed volume, particles vibrate in place
- **Liquids**: Variable shape, fixed volume, particles slide past each other
- **Gases**: Variable shape, variable volume, particles move freely
- **Energy and movement**: How temperature affects particle behavior

### **State Changes in NZ Context (20 minutes)**

- **Water cycle**: Rain, snow, steam in geothermal areas
- **Geothermal energy**: Steam from hot springs and geysers
- **Food processing**: Freezing fish, drying fruit, making ice cream
- **Weather patterns**: Cloud formation, rain, hail, snow

### **Practical Investigations (15 minutes)**

- **Melting ice**: Temperature changes during phase transitions
- **Evaporation rates**: Different liquids in NZ climate
- **Condensation**: Water vapor to liquid droplets
- **Freezing**: Making ice crystals and frost patterns

---

## 🎨 **ACTIVITIES & ENGAGEMENT**

### **Activity 1: NZ State Change Scavenger Hunt (15 minutes)**

**Materials**: Photos of NZ phenomena, state change cards
**Process**:

1. Match NZ examples to state changes (geysers → evaporation, glaciers → melting)
2. Discuss environmental factors affecting each change
3. Create visual timeline of state changes in nature

### **Activity 2: Particle Dance (10 minutes)**

**Materials**: Open space, music, student volunteers
**Process**:

1. Students act as particles in different states
2. Solid: Standing close, gentle vibration
3. Liquid: Moving around, staying together
4. Gas: Moving freely, spreading out
5. Add heat (energy) to show state changes

### **Activity 3: Geothermal Investigation (15 minutes)**

**Materials**: Hot water, cold water, thermometers, beakers
**Process**:

1. Measure temperature changes during heating/cooling
2. Observe steam formation and condensation
3. Record data and create temperature graphs
4. Connect to Rotorua geothermal activity

### **Activity 4: Weather State Changes (10 minutes)**

**Materials**: Weather maps, temperature data, cloud photos
**Process**:

1. Analyze NZ weather patterns and state changes
2. Predict state changes based on temperature
3. Discuss impact on daily life and activities
