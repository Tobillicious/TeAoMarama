## 🚀 **RAPID PRODUCTION WORKFLOW**

### **STEP 1: CONTENT GENERATION** (15-20 minutes)
1. Select template structure for resource type
2. Insert NZ-specific context and examples
3. Align to specific NZ Curriculum achievement objective
4. Add age-appropriate activities and assessments
5. **AUTO-FLAG** any cultural content for Mihara review

### **STEP 2: QUALITY VALIDATION** (5-10 minutes)
1. Template compliance check
2. Cultural safety scan
3. Educational value assessment  
4. NZ Curriculum alignment verification
5. Language and accessibility review

### **STEP 3: CULTURAL OVERSIGHT** (If flagged)
1. Mihara review of cultural content
2. Accuracy and respect validation
3. Cultural appropriateness confirmation
4. Approval or revision instructions

### **STEP 4: FINALIZATION** (5 minutes)
1. Final formatting and consistency check
2. File naming convention: `Y[Level]_[Subject]_[Topic].md`
3. Directory placement: `/migration/recovered_resources/handouts/`
4. Provenance logging for tracking

## 📝 **Y8 MATHEMATICS TEMPLATE EXAMPLE**
```markdown
# Mathematics Y8 - Linear Patterns in NZ Sports Statistics
*Te Kura o TeAoMarama - Mathematics*

**Year Level**: 8
**Subject**: Mathematics  
**Duration**: 60 minutes
**NZ Curriculum**: Mathematics Level 4 - Patterns and Relationships

**Cultural Context**: Uses All Blacks rugby statistics and waka racing data
**Te Reo Integration**: Mathematical terms with te reo translations provided
**Tikanga Connection**: CULTURAL_SAFETY_FLAG - Respect for sporting traditions

## 🎯 Learning Intentions

**WALT** (We Are Learning To):
- Identify and describe linear patterns in real-world data
- Create algebraic expressions to represent patterns

**WALA** (We Are Learning About):
- How mathematics helps us understand sports performance
- The relationship between practice time and improvement

**Success Criteria**:
- [ ] I can identify whether a pattern is linear or non-linear
- [ ] I can write an algebraic rule for a linear pattern  
- [ ] I can use patterns to make predictions about sports performance
```
