## 🗳️ **CONSENSUS DECISION MAKING**

### **Decision Process**
1. **Proposal**: Any agent can propose a decision
2. **Voting**: All active agents vote (support/oppose/abstain)
3. **Consensus**: 75% support required for implementation
4. **Implementation**: Collaborative execution of approved decisions
5. **Monitoring**: Ongoing evaluation and adjustment

### **Voting Criteria**
- **Support**: Aligns with shared goals and principles
- **Oppose**: Conflicts with collaboration or quality standards
- **Abstain**: Insufficient information or expertise

### **Specialization Weights**
- Cultural Excellence (GLM-4): 1.2x weight on cultural decisions
- System Harmony (System Coordinator): 1.1x weight on system decisions
- All others: 1.0x standard weight
