## 🎯 **Learning Objectives**

By the end of this lesson, students will be able to:

- Collect, organize, and display statistical data using appropriate graphs

- Analyze data sets with consideration for New Zealand contexts
- Make informed conclusions based on statistical evidence
- Communicate findings using mathematical language

## 🏛️ **Content Structure**

### **1. Introduction to NZ Statistics (10 minutes)**

**Real-World Context**: Understanding data in Aotearoa New Zealand

- Population demographics and trends
- Economic indicators and employment data
- Environmental statistics and climate data
- Sports performance and participation rates

**Key Concepts**:

- Primary vs secondary data sources
- Census data and government statistics
- Sample vs population in NZ context

### **2. Data Collection Methods (15 minutes)**

**Practical Applications**:

- School-based surveys (uniform preferences, lunch choices)
- Community data collection (local weather patterns)
- Sports statistics (school team performance)
- Environmental monitoring (local water quality)

**Data Collection Techniques**:

- Surveys and questionnaires
- Observations and measurements
- Secondary data sources (Stats NZ, school records)

### **3. Data Organization and Display (20 minutes)**

**Graphical Representations**:

- Bar graphs for categorical data
- Histograms for continuous data
- Line graphs for time series data
- Box plots for distribution analysis

**NZ Examples**:

- Temperature data across NZ regions
- Population growth in major cities
- Tourism statistics by region
- Educational achievement data

### **4. Statistical Measures (15 minutes)**

**Central Tendency**:

- Mean: Average calculations with real data
- Median: Middle value identification
- Mode: Most frequent value analysis

**Practical Calculations**:

- Class test scores analysis
- Heights of students in different year levels
- Travel times to school
- Screen time usage data

## 🎮 **Learning Activities**

### **Activity 1: NZ Weather Data Analysis**

**Duration**: 20 minutes
**Materials**: Weather data sheets, calculators, graph paper

**Instructions**:

1. Students receive monthly temperature data for 3 NZ cities
2. Calculate mean, median, and mode for each city
3. Create appropriate graphs to display the data
4. Compare and contrast weather patterns
5. Draw conclusions about regional climate differences

**Learning Outcomes**:

- Practice with real NZ data
- Develop graphing skills
- Understand statistical measures
- Make geographical connections

### **Activity 2: School Community Survey**

**Duration**: 25 minutes
**Materials**: Survey templates, data collection sheets

**Instructions**:

1. Design a survey about school community preferences
2. Collect data from classmates (sample size: 20-30 students)
3. Organize responses into categories
4. Create visual representations
5. Present findings to the class

**Survey Topics**:

- Preferred lunch options
- After-school activity preferences
- Transport methods to school
- Technology usage patterns

### **Activity 3: Statistical Report Analysis**

**Duration**: 15 minutes
**Materials**: NZ statistical reports, analysis worksheets

**Instructions**:

1. Read a simplified NZ statistical report
2. Identify key data points and trends
3. Evaluate the reliability of the data
4. Consider potential biases or limitations
5. Formulate questions about the data
