# Year 5 Science: Forces and Motion at the Playground

**New Zealand Curriculum Level 3**

## Learning Objectives

Students will be able to:

- Identify different forces acting on playground equipment
- Explain how forces cause objects to start, stop, or change direction
- Investigate friction, gravity, and push/pull forces
- Apply scientific thinking to everyday playground experiences

---

## Lesson Plan (60 minutes)

### Introduction (10 minutes)

**Hook Activity:** "Playground Force Detective"

Show students photos of common playground equipment:

- Swing set
- Slide
- Seesaw
- Monkey bars
- Roundabout

Ask: "What forces do you think are acting on children using this equipment?"

### Investigation Activities

#### Activity 1: Swing Investigation (15 minutes)

**Question:** What forces affect how high and fast a swing goes?

**Equipment per group:**

- String (1 meter)
- Small ball or toy
- Stopwatch
- Ruler

**Method:**

1. Make a simple pendulum with string and ball
2. Pull the ball back 10cm and release - count swings in 30 seconds
3. Pull the ball back 20cm and release - count swings in 30 seconds
4. Try with different weights

**Recording Sheet:**
| Starting Position | Number of Swings in 30s | What Force Started Motion? | What Force Stopped It? |
|------------------|--------------------------|---------------------------|------------------------|
| 10cm pull back | | | |
| 20cm pull back | | | |

**Discussion Questions:**

- What force starts the swing moving? (Push from person)
- What force brings it back? (Gravity)
- What force eventually stops it? (Air resistance/friction)

#### Activity 2: Slide Friction Test (15 minutes)

**Question:** How does surface texture affect sliding speed?

**Equipment per group:**

- Small toy car
- Different materials: aluminum foil, sandpaper, fabric, plastic wrap
- Wooden board or large book
- Stopwatch

**Method:**

1. Cover the board with aluminum foil, tape smooth
2. Place car at top, let go, time how long to reach bottom
3. Repeat with other materials
4. Test each surface 3 times

**Recording Sheet:**
| Surface Material | Time 1 | Time 2 | Time 3 | Average Time | Friction Level (High/Medium/Low) |
|-----------------|---------|---------|---------|--------------|----------------------------------|
| Aluminum foil | | | | | |
| Sandpaper | | | | | |
| Fabric | | | | | |
| Plastic wrap | | | | | |

#### Activity 3: Seesaw Balance Challenge (15 minutes)

**Question:** How do forces balance on a seesaw?

**Equipment per group:**

- Ruler (30cm)
- Pencil (fulcrum)
- 10 identical blocks or coins
- Tape

**Method:**

1. Balance ruler on pencil at the 15cm mark
2. Place 2 blocks on the 5cm mark (left side)
3. Experiment: Where do you need to place blocks on right side to balance?
4. Try different combinations

**Recording Sheet:**
| Left Side (blocks × position) | Right Side (blocks × position) | Balanced? (Yes/No) |
|-------------------------------|--------------------------------|-------------------|
| 2 blocks at 5cm | 2 blocks at 25cm | |
| 3 blocks at 10cm | 1 block at ? | |
| 1 block at 5cm | ? blocks at 20cm | |

### Playground Connection (5 minutes)

**Real Playground Application:**

"Now that we understand forces, let's think about our school playground:"

1. **Swings:** What happens if you jump off at the highest point? (Gravity pulls you down, but you keep moving forward - momentum!)

2. **Slide:** Why do you go faster on a hot, sunny day? (Less friction when slide is warm and dry)

3. **Monkey bars:** What forces help you swing from bar to bar? (Gravity pulls down, your muscles pull you forward)

4. **Roundabout:** Why do you feel pushed outward when spinning? (Centrifugal force)

## Assessment Opportunities

### Formative Assessment

- **Observation checklist** during investigations
- **Science journal entries** explaining what they learned
- **Peer discussions** about force predictions

### Summative Assessment

**Design a New Playground Equipment**

"You've been asked to design a new piece of playground equipment for our school. Your design must use at least 3 different forces we've studied."

**Requirements:**

1. Draw and label your design
2. Explain which forces are involved and how
3. Describe what makes it safe and fun
4. Write instructions for how to use it

**Success Criteria:**

- Correctly identifies at least 3 forces
- Shows understanding of how forces work
- Design is realistic and safe
- Clear explanations using science vocabulary

---

## Vocabulary

**Word Bank for Display:**

- **Force:** A push or pull
- **Gravity:** Force that pulls objects toward Earth
- **Friction:** Force that opposes motion between surfaces
- **Momentum:** The "oomph" that keeps moving objects moving
- **Balanced forces:** When forces are equal and opposite
- **Unbalanced forces:** When one force is stronger, causing movement

## Extension Activities

### For Advanced Students

1. **Investigate ramps:** How does slope angle affect rolling speed?
2. **Research project:** Find out about forces in extreme sports
3. **Engineering challenge:** Build a marble run using force principles

### For Students Needing Support

1. **Force sorting activity:** Cut out pictures and sort into "push" or "pull"
2. **Playground walkabout:** Identify forces while actually using equipment
3. **Simple demonstrations:** Teacher-led examples with clear explanations

## Safety Notes

- All investigations use safe, small-scale equipment
- Remind students about playground safety rules
- Supervise any outdoor testing activities
- Check for allergies if using different textured materials

## Resources Needed

- Investigation equipment (listed above)
- Clipboards for outdoor recording
- Digital camera for recording playground observations
- Science journals
- Playground equipment access (weather permitting)

## Cross-Curricular Links

- **Mathematics:** Measuring time, counting, basic data analysis
- **Physical Education:** Understanding body mechanics in playground activities
- **Technology:** Simple machines and levers (seesaw mechanics)
- **English:** Science vocabulary development, explanation writing

## Homework

**Family Force Hunt:** Find 5 examples of forces being used around your home. Draw pictures and write whether each is a push or pull force.

**Examples might include:**

- Opening a door (push or pull)
- Sliding a chair (push)
- Brushing teeth (push)
- Pulling socks on (pull)
- Gravity making things fall (pull)

---

## Teacher Notes

- This lesson works best on a day when you can access the playground
- Have backup indoor activities if weather doesn't permit outdoor time
- Encourage students to think about forces they use every day
- Connect to students' own experiences for better understanding
- Consider inviting a parent who works with machines/engineering to talk about forces in their job

**Learning Progression:**
This lesson builds toward Level 4 understanding of forces and prepares students for more complex physics concepts in intermediate and secondary school.
