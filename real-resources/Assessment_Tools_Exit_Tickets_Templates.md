## Digital Exit Ticket Options

### **QR Code Surveys**

Create simple Google Forms with these questions and generate QR codes for quick access:

- How confident do you feel about today's learning? (1-5 scale)
- What was the most important thing you learned?
- What questions do you still have?

### **Padlet Sharing**

Set up class Padlet walls for:

- "What stuck with you today?"
- "Questions we still have"
- "Real-world connections"

### **Flipgrid Reflections**

Video responses to:

- "Explain today's learning in 30 seconds"
- "What would you tell a friend about this topic?"
- "Show us something you learned using actions or drawings"

## Teacher Tips for Using Exit Tickets

### **Making Them Meaningful**

- ✅ Use responses to plan next day's learning
- ✅ Share anonymous responses with class to address common misconceptions
- ✅ Give specific feedback on exit tickets, not just checkmarks
- ✅ Vary the format to keep students engaged

### **Time-Saving Strategies**

- ✅ Pre-print tickets for easy distribution
- ✅ Use same format for a week then switch
- ✅ Have students keep exit tickets in learning journals
- ✅ Use digital options when possible

### **Addressing Common Challenges**

- **"I don't know" responses:** Provide sentence starters or multiple choice options
- **Rushed responses:** Build in reflection time before handing out tickets
- **Students who finish early:** Have extension activities ready
- **Consistent patterns:** Use data to inform individual and whole-class instruction

---

## Differentiated Exit Tickets

### **For Different Learning Styles**

- **Visual learners:** Include drawing/diagram options
- **Kinesthetic learners:** Use movement-based scales (stand up for understanding)
- **Auditory learners:** Allow verbal responses recorded on devices
- **Reading/writing learners:** Provide multiple text-based options

### **For Different Language Levels**

- **Emerging English:** Use pictures, symbols, or native language options
- **Developing English:** Provide sentence frames and vocabulary support
- **Fluent English:** Offer open-ended creative response options

These templates can be adapted for any subject area and year level. The key is consistency in use and genuine response to student feedback to improve learning outcomes.
