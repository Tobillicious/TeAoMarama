### **GLM Symphony Commands**
```bash
npm run glm:symphony:orchestrate  # Orchestrate full symphony
npm run glm:symphony:status       # Check symphony status
npm run glm:symphony:enhance      # Enhance content with symphony
```

### **GLM Enhancement Commands**
```bash
npm run glm:process               # Process all enhancement tasks
npm run glm:status                # Check task status
npm run glm:enhance-content       # Enhance specific content
```

### **GLM Content Generation Commands**
```bash
npm run glm:generate lesson-plan  # Generate lesson plans
npm run glm:generate assessment   # Generate assessments
npm run glm:generate activity     # Generate activities
npm run glm:report                # View generation report
npm run glm:list                  # List all generated content
```

### **GLM Workflow Automation Commands**
```bash

npm run glm:workflow:status       # Show workflow status
npm run glm:workflow:list         # List available workflows
npm run glm:workflow:report       # Generate workflow report
```
